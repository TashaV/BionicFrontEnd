# Способы подключить стиль:

  - в теле <head> прописать <link rel="styleshit" href="style.css"/>
  - в теле <head> прописать тэг <style> описание css </style>. Так же этот тэг можно прописать после <head> 
    и <body> (перед закрытием тэга </html>), но так делать не рекомендуется
  - в <body> любому элементу прописать атрибут style="color:green;". В этом атрибуте можно прописать сколько угодно
    свойств через точку с запятой
  - для изображений можно указать атрибуты (width, например) напрямую (без style)

# Структура html документа
  - <!doctype html> - указывает, что этот файл является html
  - <html>
  - <head> - метатеги, которые влияют на отображение, но сами не отображаются
  - <body> - всё, что отображается на странице

* Если в <head> подключается последовательно несколько css файлов, то последний будет иметь приоритет по отображению

* В браузере стили, установленные по-умолчанию, выделены серым фоном

# По-умолчанию блочный элемент занимает 100% пространства. Строчные элементы занимают столько места, сколько в них
  контента. Строчно-блочные элементы - пример: изображение.

# Строчные элементы лежат внутри блочных элементов. Блочные элементы не должны находиться внутри строчных элементов.

* Margin в браузере выделен оранжевым


! Ключевые моменты:

  - Валидность документа
  - Как можно меньше тэгов
  - Формат документа (соблюдать иерархию вложенности - внешний вид)
